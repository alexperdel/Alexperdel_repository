Hogar Solar

Copilot Interno

Plataforma web que automatiza procesos internos y externos de la empresa: gestion de citas comerciales, presupuestos, creacion automatizada de documentacion, todo conectado al CRM. Dos clics y se ejecuta.

Vue FastAPI Python Streamlit APIs Docker AWS

El reto

Los equipos de Hogar Solar realizaban muchas tareas manuales repetitivas: confirmar citas, generar presupuestos, crear documentacion, actualizar el CRM... Procesos que consumian tiempo y eran propensos a errores.

La solucion

Desarrollamos una plataforma interna que centraliza y automatiza procesos criticos del negocio:

  • Gestion automatizada de citas comerciales conectada al calendario y CRM
  • Generacion de presupuestos con datos precargados del cliente
  • Creacion automatica de documentacion y contratos
  • Integracion con HubSpot y sistemas internos via APIs de terceros
  • Dashboards con Streamlit para visualizacion de metricas
  • Todo en una interfaz simple: dos clics y se ejecuta

Stack tecnologico

Vue: Frontend moderno para la interfaz principal. Componentes reutilizables y UX fluida.

FastAPI: Backend en Python que orquesta toda la logica de negocio y conexiones.

Python: Scripts de automatizacion y procesamiento de datos.

Streamlit: Dashboards rapidos para visualizacion de datos y metricas internas.

APIs: Integracion con HubSpot, Google Calendar, y multiples servicios de terceros.

Docker + AWS: Despliegue en contenedores para escalabilidad y mantenimiento.

Mi rol

Lidere el diseno de la arquitectura y la definicion de requisitos con las areas de negocio. Desarrolle los primeros prototipos funcionales que luego el equipo de desarrollo escalo y profesionalizo. Coordino la evolucion del producto y las nuevas integraciones.

Resultados

La plataforma ha reducido drasticamente el tiempo dedicado a tareas administrativas. Los equipos comerciales y de atencion al cliente pueden enfocarse en lo importante mientras el sistema ejecuta los procesos en segundo plano.
