Hogarsolar

Copilot Interno

Plataforma web que automatiza procesos internos y externos de la empresa: gestión de citas comerciales, presupuestos, creación automatizada de documentación, todo conectado al CRM. Dos clics y se ejecuta.

Vue FastAPI Python Streamlit APIs Docker AWS

El reto

Los equipos de Hogarsolar realizaban muchas tareas manuales repetitivas: confirmar citas, generar presupuestos, crear documentación, actualizar el CRM... Procesos que consumían tiempo y eran propensos a errores.

La solución

Desarrollamos una plataforma interna que centraliza y automatiza procesos críticos del negocio:

  • Gestión automatizada de citas comerciales conectada al calendario y CRM
  • Generación de presupuestos con datos precargados del cliente
  • Creación automática de documentación y contratos
  • Integración con HubSpot y sistemas internos via APIs de terceros
  • Dashboards con Streamlit para visualización de métricas
  • Todo en una interfaz simple: dos clics y se ejecuta

Stack tecnológico

Vue: Frontend moderno para la interfaz principal. Componentes reutilizables y UX fluida.

FastAPI: Backend en Python que orquesta toda la lógica de negocio y conexiones.

Python: Scripts de automatización y procesamiento de datos.

Streamlit: Dashboards rápidos para visualización de datos y métricas internas.

APIs: Integración con HubSpot, Google Calendar, y múltiples servicios de terceros.

Docker + AWS: Despliegue en contenedores para escalabilidad y mantenimiento.

Mi rol

Lideré el diseño de la arquitectura y la definición de requisitos con las áreas de negocio. Desarrollé los primeros prototipos funcionales que luego el equipo de desarrollo escaló y profesionalizó. Coordino la evolución del producto y las nuevas integraciones.

Resultados

La plataforma ha reducido drásticamente el tiempo dedicado a tareas administrativas. Los equipos comerciales y de atención al cliente pueden enfocarse en lo importante mientras el sistema ejecuta los procesos en segundo plano.
