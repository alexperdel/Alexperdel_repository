# Alex Perdel - Portfolio Personal

Sitio web personal y portfolio de Alex Perdel, Head of AI & Automation.

## Descripcion

Este repositorio contiene el codigo fuente de mi sitio web personal, donde muestro mi experiencia profesional, proyectos y habilidades en el campo de la Inteligencia Artificial y Automatizacion.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- SEO optimizado con Schema.org

## Estructura del proyecto

```
/
├── index.html          # Pagina principal
├── sobre-mi.html       # Pagina sobre mi
├── proyectos/          # Seccion de proyectos
├── css/                # Estilos
├── js/                 # Scripts
├── assets/             # Recursos (imagenes, etc.)
├── robots.txt          # Configuracion para crawlers
└── sitemap.xml         # Mapa del sitio
```
